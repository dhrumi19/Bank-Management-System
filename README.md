🏦 Bank Management System

A comprehensive desktop-based Banking Management System developed using Core Java, Java Swing, and Object-Oriented Programming (OOP) principles. The application provides a secure and user-friendly platform for managing customer accounts, performing banking transactions, generating reports, and maintaining persistent records through file handling and serialization.

The project simulates real-world banking operations, allowing users to perform deposits, withdrawals, fund transfers, account management, and transaction tracking while ensuring data integrity and security.
